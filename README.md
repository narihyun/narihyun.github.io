## 문제 정의

### "친구랑 같이 살면 무조건 싸운다?"
기숙사나 쉐어하우스에서 공동생활을 하는 사람들은 **물품 부족, 비용 정산, 가사 분담**으로 인한 생활적인 불편과 감정적인 어색함을 경험합니다.

#### 1. 조용한 눈치게임
- 누군가는 세제를 여러 번 대신 사고, 누군가는 청소 차례를 놓치지만, 서로 얼굴을 보며 "이번엔 네가 해"라고 직설적으로 말하기는 쉽지 않습니다.
- 참다가 말하면 '예민한 사람' 취급을 받고, 가만히 있자니 나만 '손해 보는 기분'이 듭니다.

#### 2. 기존 시스템의 한계
- **카카오톡**: 대화에 밀려 약속이 휘발되고 기록을 찾기 어렵습니다.
- **구글 시트**: 관리가 번거로워 업데이트가 불규칙해집니다.
- 결국 생활의 투명성이 떨어지고, **신뢰 관계가 약화**됩니다.

## 솔루션

### "하기 힘든 말은 시스템이 대신 합니다"
**RoomEase**는 룸메이트들이 입주 초기에 구체적인 생활 약속 (Living Agreement)을 함께 만들고, 이를 시스템적으로 관리할 수 있게 도와줍니다.

#### 핵심 가치
- **약속 표준화**: '화장실 청소', '쓰레기 배출' 등 갈등이 잦은 항목을 템플릿화하여 누락 없이 공평한 약속을 만듭니다.
- **간편한 시작**: 복잡한 설치나 로그인 없이 **링크 하나로 3분 만에** 합의된 약속을 만들 수 있습니다.
- **투명성**: 누가, 언제, 무엇을 해야 하는지 명확히 시각화하여 불필요한 눈치 게임을 없앱니다.

## 주요 기능

### 1. 5단계 온보딩 플로우

**Step 1: 집 정보 입력**
- 사용자 이름과 집 이름을 설정
- 고유 House ID 자동 생성 (타임스탬프 기반)

**Step 2: 룸메이트 추가**
- 함께 사는 룸메이트 이름 입력
- 랜덤 아바타 자동 배정
- 수정/삭제 기능 제공

**Step 3: 규칙 설정**
- **템플릿 제공**: 화장실 청소, 쓰레기 배출, 설거지, 분리수거, 빨래 등
- **주기 설정**: 매주, 격주, 매달
- **요일 지정**: 월~일 선택
- **담당 방식**:
  - **돌아가면서**: 자동 로테이션
  - **고정**: 특정 담당자 지정

**Step 4: 메모 추가** *(선택사항)*
- 우리 집만의 특별한 에티켓 기록

**Step 5: 완료 및 공유**
- 완성된 약속서 확인
- 공유 링크 생성 및 복사

### 2. 데이터 보관

- **서버리스**: Google Sheets를 데이터베이스로 활용
- **실시간 동기화**: 변경 사항이 즉시 모든 사용자에게 반영

### 3. 공유 및 공동 편집

- **URL 파라미터 기반**: `?houseId=xxxxxxxxxx` 로 집 데이터 공유
- **회원가입 불필요**: 링크만으로 즉시 접속
- **공동 편집**: 모든 룸메이트가 규칙 추가/수정 가능

## 시작하기

### 사전 요구사항 (Prerequisites)

#### 1. Node.js 설치
```bash
# 버전 확인
node -v  # v18.0.0 이상 권장
npm -v   # v9.0.0 이상 권장
```

**설치 방법:**
- **Mac**: `brew install node` 또는 [nvm](https://github.com/nvm-sh/nvm) 사용
- **Windows**: [Node.js 공식 웹사이트](https://nodejs.org/)에서 LTS 버전 다운로드

#### 2. Git 설치
```bash
# 버전 확인
git --version
```

**설치 방법:**
- **Mac**: `brew install git`
- **Windows**: [Git 공식 웹사이트](https://git-scm.com/)

---

### 설치 방법 (Installation)

#### 1단계: 저장소 클론
```bash
git clone https://github.com/narihyun/roomease.git
cd roomease
```

#### 2단계: 의존성 설치
```bash
npm install
```

---

### 실행 방법 (Running)

#### 개발 서버 실행
```bash
npm run dev
```

**실행 결과:**
```

➜  Local:   http://localhost:3000/
➜  Network: use --host to expose
```

브라우저에서 터미널에 표시된 로컬 주소(예: `http://localhost:3000/`) 접속

## 사용 방법

### 시나리오 1: 새로운 집 만들기 (Create)
1.  **시작하기**: 메인 화면에서 "3분 만에 우리 방 약속 만들기" 버튼을 클릭합니다.
2.  **정보 입력**: 내 이름(예: 루미)과 집 이름(예: 행복하우스)을 입력합니다.
3.  **룸메이트 초대**: 함께 사는 룸메이트의 이름(예: 지수)을 입력하고 추가합니다.
4.  **규칙 설정**:
    *   추천 템플릿(예: 화장실 청소)을 선택하거나 직접 입력합니다.
    *   주기(매주 월요일), 담당 방식(돌아가며/고정)을 설정하고 추가합니다.
5.  **완료**: 모든 설정이 끝나면 약속 확정하기를 누릅니다.

### 시나리오 2: 공유 및 공동 편집 (Share & Edit)
1.  **링크 공유**: 완료 화면에서 **"초대 링크 복사"** 버튼을 누릅니다.
2.  **접속**: 복사한 링크를 카카오톡 등으로 공유하고, 룸메이트가 접속합니다.
3.  **자동 동기화**: 별도의 로그인 없이 저장된 집 정보와 규칙을 즉시 확인할 수 있습니다.
4.  **수정 하기**:
    *   "규칙 수정하기"를 눌러 룸메이트가 직접 규칙을 추가하거나 담당자를 변경할 수 있습니다.
    *   변경 사항은 자동으로 Google Sheets에 저장되며, 실시간으로 반영됩니다.
